### 2.svg 常用的标签说明

#### （1）g 字符

```
<g>标签SVG元素是用于组其他SVG元素的容器。
应用于<g>元素的转换将在其所有子元素上执行，并且其任何属性都由其子元素继承。

```

#### （2）defs

```
SVG允许定义图形对象以备后用。建议尽可能在元素内部定义引用的<defs>元素。

```

#### （3）- clippath

```
<clipPath>SVG元素定义剪切路径。使用clip-path属性使用/引用剪切路径。
纠错
剪切路径限制了可以应用paint的区域。从概念上讲，绘图中位于由当前活动剪切路径限定的区域之外的任何部分都未绘制。

```
